# Screenshots

Binary screenshots are not committed in this repository.

To regenerate listing screenshots locally:

1. Start the app: `npm run dev -- --host 0.0.0.0 --port 4173`
2. Open `http://localhost:4173/src/sidepanel/index.html`
3. Capture and save PNGs with these names in this folder:
   - `side-panel.png`
   - `tab-detection.png`
   - `prompt-editor.png`
   - `chatgpt-run.png`
   - `debug-logs.png`
