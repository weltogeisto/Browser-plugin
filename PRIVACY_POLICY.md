# Privacy Policy

**Effective Date:** March 9, 2026

## Data Collection

Model Judge MVP does not collect, store, transmit, or share any user data.

## What We Do NOT Collect

- No personal information
- No browsing history
- No API keys or credentials
- No usage analytics or telemetry
- No crash reports

## Local Storage Only

All data (prompt templates, user preferences) is stored locally in your browser using Chrome's storage API. This data never leaves your device.

## Text Selection

When you select text and click "Refresh selection + tabs," the extension captures only the selected text from the current page. This text:
- Stays in your browser's local memory
- Is never transmitted to our servers (we have no servers)
- Is only sent to AI provider tabs YOU have already opened and authenticated

## Host Permissions

The extension requests permission to access:
- chatgpt.com
- claude.ai
- perplexity.ai

These permissions are used ONLY to detect open tabs and automate text input into tabs you have already authenticated with these services.

## Third-Party Services

This extension automates interaction with third-party AI services (ChatGPT, Claude, Perplexity). Your interactions with those services are governed by their respective privacy policies:
- [OpenAI Privacy Policy](https://openai.com/policies/privacy-policy)
- [Anthropic Privacy Policy](https://www.anthropic.com/privacy)
- [Perplexity Privacy Policy](https://www.perplexity.ai/privacy)

## Updates to This Policy

We will update this policy if our data practices change. Check this document periodically.
